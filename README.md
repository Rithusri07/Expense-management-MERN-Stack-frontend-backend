# 💰 Expense Management System

## 📌 Overview
A full-stack expense tracking application built using the MERN stack.  
It enables users to securely manage their personal finances by tracking income, expenses, and receipts with an efficient and scalable backend.

---

## 🚀 Features
- 🔐 User Authentication using JWT and bcrypt  
- 📊 Add, update, delete, and view expenses (CRUD operations)  
- 🧾 Upload receipt images using Multer  
- ⚡ Optimized database queries with MongoDB indexing  
- 🔍 Efficient filtering and aggregation of expenses  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- HTML, CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  

### Tools & Libraries
- JWT (Authentication)  
- bcrypt (Password hashing)  
- Multer (File uploads)  
- Postman (API testing)  

---

## ⚙️ How It Works

1. User registers and logs in securely using JWT authentication  
2. User can create, update, and delete expense entries  
3. Receipt images are uploaded and stored via backend  
4. MongoDB handles data storage with indexed queries for faster performance  
5. Data is fetched via REST APIs and displayed in the frontend
